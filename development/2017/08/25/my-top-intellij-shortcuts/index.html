Meus atalhos mais usados do IntelliJ

por José Victor Alves de Souza em 25 de Agosto de 2017
3 minutos de leitura

No dia a dia de nós desenvolvedores, os atalhos são bastante úteis pois agilizam o nosso trabalho e portanto nos trazem mais produtividade. Além disso, convenhamos que ficar intercalando entre teclado e mouse é bastante cansativo! Assim, conhecer os atalhos do seu ambiente de desenvolvimento é fundamental e vai te economizar muito tempo e esforço!

Trabalho com desenvolvimento em Java e por isso uso o IntelliJ durante a maior parte do tempo. Mas independente da IDE ou do editor de código que você use, busque sempre decorar o máximo de atalhos possíveis. A ideia realmente é usar o mouse o mínimo possível!

Uma dica legal para quem está começando é decorar os atalhos de ações que você costuma realizar com maior frequência, como por exemplo, abrir uma classe ou um arquivo do seu projeto atual, refatorar um método ou um atributo, etc., e a medida que você for se acostumando com uso desses atalhos você vai incremetando, tentando memorizar novos.

A seguir listei 27 atalhos que mais uso no IntelliJ:

1. ctrl + shift + n - Abrir uma classe ou um arquivo do seu projeto atual

2. ctrl + tab - Alternar entre as abas abertas

3. shift + f6 - Renomear métodos, variáveis, arquivos e etc.

4. alt + j - Multicursor, permite alterar múltiplas ocorrências ao mesmo tempo

Fonte : https://blog.jetbrains.com/clion/2016/07/become-multi-armed-with-clions-multiple-cursors/

5. ctrl + d e ctrl + y - Duplicar e deletar uma linha, respectivamente

6. alt + f7 ou ctrl + alt + 7 no Linux - Navegar por usos da ocorrência

7. ctrl + alt + v - Atribui o resultado da chamada um método a uma variável (extrair variável)

8. ctrl + shift + f - Pesquisa um termo ou uma expressão em todos os arquivos do projeto

9. crtl + shift + r - Pesquisa e substitui um termo ou expressão em todos os arquivos do projeto

10. alt + f8 (in debug mode) ou ctrl + alt + 8 no linux - Avaliar/Executar expressão em debug

11. ctrl + / - Comentar linhas seleciondas

12. ctrl + h - Navegar na hierarquia de heranças da classe atual

13. ctrl + alt + b - Ir para a implementação do método / classe

14. ctrl + u - Ir para o método da classe pai

15. ctrl + shift cima / baixo - Mover linhas ou blocos de código para cima / baixo

16. alt + 1/2/3/4/5/6/7/8/9 - Exibe e mantém o foco nas janelas auxiliares (projeto, debug, controle de versão, etc.)

17. alt + f12 - Exibe o terminal

18. alt + arrows up/down - Navega para o método seguinte ou anterior

19. alt + arrow lef/right - Alterna entre abas aberta anterior e posterior

20. ctrl + Numpad + e ctrl + Numpad - - Expande e recolhe um bloco de código, respectivamente

21. ctrl + g - Navega até a linha

22. f2 - Navega até o próximo erro do arquivo atual

23. ctrl + f4 - Fecha a aba atual

24. shift + f10 e shift + f9 - Executa e inicia o Debug do projeto respectivamente

25. ctrl + f2 - Para execução do projeto

26. ctrl + shift + f12 - Esconde todas as janelas auxiliares, deixando só a de edição de código ativa

27. ctrl + shift + a - Executa uma ação ou opção da IDE pelo nome, caso você desconheça o comando
